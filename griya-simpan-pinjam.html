📖 Panduan Sistem GSP

Logika Bisnis GSP (Diperbarui)
Sistem ini berfokus murni pada operasional simpan pinjam. Tidak ada koneksi server luar, 100% data tersimpan di browser perangkat Anda untuk privasi maksimal.
📊
Definisi Dashboard
Sisa Piutang Berjalan: Murni sisa tagihan dari nasabah aktif.
Total Profit (PnL): Uang masuk yang telah nyata melebihi pokok (Realisasi). Bunga yang belum dibayar tampil sebagai Estimasi.
👥
Anggota vs Non-Anggota
Anda bisa membedakan status nasabah saat mendaftar. Gunakan Filter di menu Nasabah dan cetak CSV untuk laporan spesifik.
⚙️
Backup JSON Berkala
Sangat disarankan untuk rutin melakukan Export Data di menu Pengaturan seminggu sekali untuk mencegah hilang data jika browser ter-clear cache.
🔒
Keamanan Layar Auto-Lock
Untuk menjaga privasi, layar akan otomatis terkunci jika Anda meninggalkan perangkat (tidak ada aktivitas mouse/keyboard) selama 1 jam.
